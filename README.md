# Install the application #

Our public git :

https://github.com/CadenetVincent/node_curse

Install the application :

git clone https://github.com/CadenetVincent/node_curse
cd node_curse
npm install
npm start

Build and start the application : 

!!!!!!! IMPORTANT !!!!!!!!

if you want to start the good project with LevelDB :

npm run starttwo
